The list

10の改善

タップで開く。それぞれ「何を直すか → なぜか → やりがちな例 → 対策」の順。上のボタンで絞り込める。

直すこと
  • 喧嘩や真面目な話のときに、感情的な言葉や態度を出さない
  • 怒りや焦りが出ても、相手を傷つける言い方をしない
なぜ

感情的になると、不必要な発言や態度が出て、話し合いがさらに悪化してしまうため。

やりがち
  • 「おまえ」
  • 「無理」
  • 「ダメ」
  • 「もういい」
  • 不機嫌な態度、投げやりな返事
対策
  • イラッとしたら、すぐ返さず「考える」と言って10秒黙る
  • 感情が強くなったら「10分だけ落ち着く時間がほしい」と伝える
直すこと
  • 勝ち負けや正しさにこだわりすぎない
  • 自分の非を認めることを、負けだと思わない
  • 相手の気持ちより、自分の正しさを優先しない
なぜ

プライドが邪魔をすると、にこの本音を受け止められず、信用を失ってしまうため。

やりがち
  • ゲームなどで絶対に勝とうとする
  • 話し合いで「でも」「いや」「違う」から入る
  • 自分の悪かった点より、言い訳や説明を先にする
対策
  • 話し合いでは、まず自分の悪かった点を1つ言う
  • 「でも」「いや」「違う」から話し始めない
  • ゲームでも、勝つことより一緒に楽しむことを優先する
直すこと
  • 相手を突き放す言葉、否定されたと感じさせる言葉を使わない
  • 内容を伝える前に、言い方で傷つけない
なぜ

強い言葉を使うと、内容以前に、にこが自分自身を否定されたように感じてしまうため。

やりがち
  • 「無理」
  • 「ダメ」
  • 「違う」
  • 「意味ない」
  • 「だから言ったじゃん」
対策
  • 「無理」→「今のままだと少ししんどい」
  • 「ダメ」→「こうしてくれると助かる」
  • 「違う」→「俺の認識では少し違っていて」
  • 重い内容のLINEは5分置いて読み返す
直すこと
  • 自分の考えや判断が、常に正しいと思い込まない
  • 記憶が曖昧なことや、不確かなことを断言しない
なぜ

過信して間違えたり失敗したりすると、相手からの信頼を失ってしまうため。

やりがち
  • 「絶対こうだと思う」と決めつける
  • 曖昧な記憶なのに断言する
  • 指摘されても、すぐに認めない
対策
  • 断言する前に「自分の認識では」「違っていたらごめん」
  • 指摘されたら、まず最後まで話を聞く
  • 間違っていたら「確かに俺の認識が違っていた」と認める
直すこと
  • 小さなことでもごまかさず、正直に伝える
  • 過去に話したことと、今話すことにズレを出さない
  • 自分を守るための、その場しのぎの説明をしない
なぜ

一貫性がないと嘘だと感じさせ、信用を失ってしまうため。小さなごまかしでも、積み重なると「他のことも隠しているのでは」と不安にさせてしまうため。

やりがち
  • 予定が変わったのに伝えない
  • まだ家にいるのに、外にいるように伝える
  • 記憶が曖昧なのに断言する
  • 怒られたくなくて、話を少し変える
対策
  • 予定が30分以上変わったら、一言連絡する
  • 聞かれたことには、現在の状況を正確に答える
  • 曖昧なときは「覚えてない」「たぶん」と正直に言う
  • 間違って話したと気づいたら24時間以内に訂正
直すこと
  • 過去の出来事や、疑われそうな内容を例え話に使わない
  • 事実と関係ない、誤解されにくい例を使う
なぜ

事実と関係ありそうな例え話をすると、本当にあったことのように聞こえて、不安にさせてしまうため。

やりがち
  • 異性関係の例え話
  • 浮気や隠し事に関係する例え話
  • 過去に疑われた内容と近い例え話
  • 実際の出来事と混ざりやすい話
対策
  • 異性・嘘・隠し事・浮気に関わる例え話は避ける
  • 例える前に「これは完全に例えだけど」と伝える
  • 誤解されたら「今の例えはよくなかった」とすぐ訂正
直すこと
  • にこが不安や嫌だったことを話したときに、すぐ否定しない
  • 自分の基準や一般論で、にこの気持ちを片付けない
なぜ

否定されると、にこがこれまで頑張ってきたことや我慢してきたことまで、認められていないように感じてしまうため。

やりがち
  • 「考えすぎ」
  • 「普通は〜」
  • 「そんなことで？」
  • 「気にしすぎじゃない？」
対策
  • まず「そう感じたんだね」と受け止める
  • 反論や説明の前に、にこの気持ちを1文で要約する
  • 話の途中で遮らず、最後まで聞く
直すこと
  • 事実や理由の説明だけで終わらせず、気持ちを理解しようとする
  • 正しさよりも、まず安心して話せる雰囲気を作る
なぜ

にこにとっては、事実や理由の説明よりも、理解しようとする姿勢が安心につながるため。

やりがち
  • すぐに理由を説明する
  • 正論で返す
  • 「でも俺はこうだった」と先に自分の話をする
  • 気持ちより、事実確認を優先しすぎる
対策
  • 「それは不安になったよね」と気持ちを受け止める
  • 「そう感じさせてしまったならごめん」と伝える
  • 話し合いの最後に「話してくれてありがとう」と伝える
直すこと
  • にこが謝るまで待つのではなく、自分から向き合う
  • 喧嘩後に放置せず、安心感を戻す行動をする
なぜ

放置すると不安や不信感が大きくなり、関係がさらに悪化してしまうため。にこだけが謝ったり歩み寄ったりしているように感じさせてしまうため。

やりがち
  • にこが謝るまで自分から動かない
  • 不機嫌なまま黙る
  • 話し合いを先延ばしにする
  • 「もういい」と終わらせる
対策
  • 喧嘩後6時間以内に、自分から一度声をかける
  • 自分の悪かった点を1つ伝える
  • すぐ話せないときは「無視したいわけじゃない。落ち着いたら話したい」と伝える
  • 遅くても翌日中には話し合う
直すこと
  • 大きな話し合いのときだけでなく、普段から信用を積み上げる
  • 不安にさせる前に、予定や状況を共有する
なぜ

日常の小さな安心が積み重なることで、信頼を少しずつ戻せるため。

やりがち
  • 予定が変わったのに伝えない
  • 帰宅時間が大きく変わっても連絡しない
  • 感謝を言葉にしない
  • 不安にさせた後だけ優しくする
対策
  • 予定が変わったら先に共有する
  • 帰宅時間が30分以上変わる場合は連絡する
  • 1日1回は「ありがとう」を伝える
  • 週1回、10〜15分だけ生活や気持ちの振り返りをする
Rephrase

言い換え表

とっさに出そうな言葉を、そのまま置き換えるための表。同じ内容でも、言い方だけで伝わり方が変わる。

出そうになる言葉こう言い換える
無理→今のままだと少ししんどい
ダメ→こうしてくれると助かる
違う→俺の認識では少し違っていて
絶対こうだと思う→自分の認識では〜。違っていたらごめん
考えすぎ／気にしすぎ→そう感じたんだね
普通は〜／そんなことで？→それは不安になったよね
でも／いや（出だし）→まず自分の悪かった点を1つ言う
もういい→無視したいわけじゃない。落ち着いたら話したい
だから言ったじゃん→話してくれてありがとう
おまえ→にこ
（曖昧なのに断言）→覚えてない／たぶん、と正直に言う
（黙る・不機嫌な態度）→考える、と言って10秒おく
